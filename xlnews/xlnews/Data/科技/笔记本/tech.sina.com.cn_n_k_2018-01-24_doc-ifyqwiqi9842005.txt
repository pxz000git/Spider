　　本文来自IT之家　　今天，微软发布了Windows Server Insider Preview Build 17079。有趣的是，一周前，微软发布了Build 17074，而通常情况下，服务器版Windows一个月才会更新一次。　　有一点值得注意的是，在Windows Server Redstone 4预览版中，将不再需要像以前的更新一样使用产品密钥，现在，系统的激活将包含在Build中，Insider Preview Build 17079将于今年7月2日到期，届时，你必须升级到新版本。　　以下是Windows Server Build 17079的新功能：　　：客户现在可以通过“就地升级操作系统”的方式将他们的Windows Server Version 1709安装/升级到当前预览版本。此功能从Build 17079开始可用。此功能还使客户能够执行下一个发布的预览版本的构建到构建的升级。　　运行setup.exe时，升级向导将向客户提供“保留个人文件和应用程序”的选项。这将保证Windows Server升级完成后，应用程序、数据和用户设置得以保留。　　：优化的服务器核心容器基础映像（体积减小30%）。　　MSMQ现在安装在Server Core容器中。　　关键应用程序兼容性错误修复。　　同时，这个版本的Windows Server Insider Preview还修复了大量的已知问题。
