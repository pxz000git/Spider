　　Cherry樱桃在去年8月底发布了旗下的MX BOARD新品5.0机械键盘，其最大的特色是可以自定义的人体工程学设置，配备了超大、可独立调节的掌托与独特的支脚搭配，可提供9个不同的姿势状态，以实现最大的舒适度。这款Cherry MX BOARD 5.0键盘笔者已经使用了一段时间，输入体验可谓接近完美，而且掌托的加入使得我们需要重新审视键盘在舒适度方面还能有如此契合的满足点。　　Cherry MX BOARD 5.0机械键盘的外包装十分丰富，包装正面左侧是大大红色Cherry樱桃图标Logo，右侧是干练的MX BOARD 5.0标识。“MX”相信大家都知道代表着什么，即目前指代大多数机械键盘所采用的轴体，包括黑轴、茶轴、红轴、青轴等都是MX轴。下面的“HS”是“High Speed”技术的简称。　　将黑色盒子中的包含键盘部分抽取出来则是另一个内盒，外观使用了非常鲜艳的樱桃红色，仔细看上面有大大的樱桃的印记Logo。里面分为两层，上面一层是Cherry MX BOARD 5.0键盘本体，下面是附带的键盘掌托，掌托部分后面会单独说明。　　将Cherry MX BOARD 5.0键盘取出，就是下面的样子了。键盘的重量略重，单手持握比较吃力，但也从侧面说明MX BOARD 5.0用料比较充足，据悉键盘在内部还增加了一块稳固钢板，在摆放上更显稳重。　　Cherry MX BOARD 5.0键盘采用铝制金属外壳边框，质感十足，起初笔者以为外圈是塑料材质，不过细细摸上去就会感觉到铝制金属的稍稍冰凉感。铝制金属外壳边框经过了细磨砂表面处理，并且带有防油涂层，不容易沾染手印或指纹污渍等。在整个键盘的外壳边框边角，MX BOARD 5.0使用的是圆润的弧角设计。　　Cherry MX BOARD 5.0键盘背面有八个红色的脚垫，其中6个固定的，2个是可以活动的，并且采用转轴的方式能够调节键盘与桌面之间的倾斜角度。除此之外，键盘的靠近中间背面内置了USB键盘线，在连接线的导出方面，键盘里分为三条凹槽供采用不同的选择，其实大家直接使用默认的直出线凹槽就比较好，不过具体情况请视使用环境而定。　　具体到键盘连接线上，相比起MX BOARD 1.0采用的橡胶线，MX BOARD 5.0采用的是更加紧凑清爽的编织线，键盘线的长度完全不用担心过短，键盘的另一端采用的USB 2.0接口，接口处材质坚硬结实，可以很好的拿捏，USB端口进行了镀金处理，呈现金色，看起来比较高档，质感也好上很多。　　　　笔者此前使用的是Cherry去年中期发布的MX BOARD 1.0 TKL机械键盘，即没有小键盘，相比起MX BOARD 1.0键盘，MX BOARD 5.0机械键盘的档次更高，键盘按键排列更加紧凑。除了拥有完整的108键之外，这款键盘还配备了人体工程学技术的键盘掌托，体验下来发觉此乃MX BOARD 5.0键盘的锦上添花。　　在开箱的时候，本着好奇的心态不断发掘，在键盘之下发现了隔层，里面就是这块大大的掌托了，其长度要比MX BOARD 5.0键盘还要长一小块，这个设计并不难猜到，是为了让手掌有更好的活动空间并依然处于掌托之上，舒适空间范围有保证。　　由于号称掌握了不少姿势，因此这款键盘掌托底部配备了四个可转动调节的垫脚，并且都是长条形，有利于稳定掌托的支撑状态。就重量上而言，这款掌托依然“用料”满满，单手持握也不轻松，因此将其摆放到桌面上并和键盘结合后，最好就不要随意调节换动了，接下来保持可靠的使用性就可以了。　　笔者比较喜欢的是，将掌托的前部调节到翘高，而下部采用默认的平底支撑，形成一定的倾斜弧度。值得注意的是，键盘和掌托之间比较容易扣合，当扣合之后，键盘的高度也会相应的提高，是否舒适大家自行调节到满意状态，就笔者而言，这种调节并不复杂，凭借你的感觉就可以快速实现。　　在材质上，MX BOARD 5.0的掌托采用塑料材料，表面采用了类肤质材质，涂层细腻舒适，但就一段时间的使用来看，掌托表面还是比较容易沾灰和污渍的，大家在使用时请多注意手掌或手指的清洁，也要及时对掌托部分进行清理，保持良好的习惯。　　上面提到MX BOARD 5.0的掌托采用类肤材质，亲身体验后便欲罢不能，除了掌托本身带来的支撑轻松感，手掌对掌托表面的感觉非常细腻舒爽，有时候会情不自禁地将手掌或手指靠在掌托上摩挲。　　在宽大的掌托右侧有很大的“MX”Logo设计，被上下括号形状线条包裹着，这里的MX Logo在视觉上很淡，不仔细看是不容易被察觉到，这里使用相机拍下了其全貌和特写。　　不过这款掌托在从包装里取出的时候，会有一定的塑料性味道，大家可以先放置掌托一段时间消散一下气味再作使用。　　　　一款机械键盘是否美观可能是一千个人眼中有一千个哈姆雷特，但键盘的最本质工作还是打字或游戏操控等输入操作，特别是对于经常使用电脑的人士来说至关重要。　　Cherry MX BOARD 5.0配备白色背光，可调节亮度并实现不同灯效，另外这款键盘采用独特的Cherry游戏字体，你会发现键帽上的字体与普通键盘有明显区别，键盘采用双色PBT键帽，保证了键帽的耐磨性。同时键帽上带有防油涂层，从使用效果上看非常不错，几乎不易粘上指纹等。键盘具有很好的亲肤质感，并且键盘的透光会更均匀。　　Cherry MX BOARD 5.0键盘背光的亮度拥有多个梯度可调，特效有常亮、呼吸、心跳三种。按下FN+F5可以打开和关闭背光，使用FN+F6/F7调节亮度，FN+F12可以调节背光特效等。　　Cherry MX BOARD 5.0的键盘上采用了独特的Cherry樱桃游戏字体，据悉该樱桃游戏字体是和德国Typemates共同研发，每一个字母和数字都经过了特殊的设计，字体显得纤细，并且都采用了开口的设计，稍显圆润，美观度还是不错的，毫无违和感，重要的是新的游戏字体并不影响大家的辨识度，这一点尤其值得称赞。　　键盘的右上角是4枚多媒体按键，另外右上角外侧分别也有四枚指示灯，樱桃Logo对应为红色，后三个为白色。指示灯比较小，上面标注数字、字母，1代表数字区开关，C代表字母大小写开关，S代表Scroll Lock屏幕滚动锁定开关。和MX BOARD 1.0类似，MX BOARD 5.0的方向键区域同样做了钻石面切割设计，起到点缀的效果。　　笔者使用的这款MX BOARD 5.0键盘是红轴，熟悉机械键盘的朋友都知道，红轴用起来没有段落感，手感比较轻盈，敲击时没有段落感，直上直下，触发键程为2.0mm，敲击时更加轻松，能很好兼顾游戏和打字的使用需求。特别适合长时间打字的编辑工作。MX BOARD 5.0键盘在大键位方面采用卫星轴稳定方式，提供了非常不错的稳定效果，不会出现左右摇晃的问题。　　在游戏方面，MX BOARD 5.0键盘反应速度很快，红轴也特别适合FPS玩家使用，比如《CS：GO》、《绝地求生》和《使命召唤》系列游戏。　　　　MX BOARD 5.0机械键盘的设计很温和，但又不会太低调，非常适合日常办公、家居等环境使用。充满人体工程学设计的掌托也赋予了这款MX BOARD 5.0机械键盘新的亮点，是不可或缺的组成部分，值得大家去细细体会。
