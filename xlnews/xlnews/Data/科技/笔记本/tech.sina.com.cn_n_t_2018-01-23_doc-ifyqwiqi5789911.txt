　　现如今，移动办公已经成为常态，轻薄本的竞争也是如火如荼，各家PC厂商都在纷纷推出自己的轻薄本新品，为了激烈的市场竞争中能够占得一席之地。而宏碁蜂鸟系列向来以极度纤薄、极其方便而著称，这次出产的AcerSwift 5仍然引来了众多惊异的目光，它依然延续了Swift系列轻薄的设计理念，并且在身形和重量上有着非常大的突破。今天我们就从Acer Swift 5的身材说起，一起探寻它轻巧的身材内到底蕴含着哪些秘密。　　　　Acer Swift 5的机身仅为970g，只相当于两瓶矿泉水的重量。而它的尺寸329mm×228mm×14.9mm，比一本书的厚度还要薄一些。这样的身材在笔记本当中可以说是翩若惊鸿了。而它轻薄的秘密就在于它的材质，它采用的是镁锂合金制作而成，它的重量约为铝合金的四分之一，但是却富有陶瓷质感，是生产轻薄本的优选材料。镁锂在周期元素表中比较靠后是非常活泼的金属材料，Acer通过强大的制造工艺把活泼金属融合在轻薄本中，可见Acer的制造实力。而Acer Swift 5的配色则采用是暮光蓝，这样的颜色使整部机器显得是沉静、简约、极富科技质感，可以与商务人士形成完美的搭配。　　Acer Swift 5准备将简约风格进行到底，所以A面的设计别无他物，纯净的蓝色面板上中间只有宏碁最具识别的logo，让人觉得它非常沉稳、内敛、大气。　　而简约的A面不同的是，当Acer Swift 5的屏幕映入你的眼帘之后，你会产生些许惊异的感觉。Acer Swift 5配备的是一块14寸全高清多指触控IPS屏幕，分辨率为1920×1080，拥有74%的NTSC色域，并且支持多点触控，也是窄边框设计，另外可以进行180度开合。同时这款屏幕还配备了Acer Color Intelligence TM 超显色技术和Acer BluelightShield TM蓝光护盾技术，让Acer Swift 5拥有更加绚丽的色彩，同时也能保护双眼健康。　　我们再来看一下Acer Swift 5的键盘。这款键盘采用的是蓝底金字的配色，为整台电脑增加了些许华贵气质。同时采用了白色背光灯，方便在黑暗环境下输入。另外，键程设计也比较适中，反馈效果也比较好，能让你充分享受输入的快感。　　Acer Swift 5的D面则延续了A面的简约风格，上边是它的进风口，下边则是它的扬声器。值得一提的是，这台机器配备了Acer TrueHarmony TM 和声至臻音效系统和高级版的杜比音效，让你觉得声临其境，天声震撼。　　为了让轻薄笔记本拥有更好的外形，在拓展接口方面做了很大的取舍，为很多用户带来了非常大的不便。但是Acer Swift 5在如此轻巧的身形下，也依然保持着比较好的拓展性。它拥有1个Type-C接口、1个HDMI接口、1个3.5mm耳机接口和2个USB3.0接口，此外USB3.0接口还具有关机充电功能。这些的拓展设计足以满足日常办公所需。　　　　　　我们使用了CPU-Z这款软件收集了Acer Swift 5的处理器信息，这台机器所搭载的是Intel Core i5-8250U处理器，属于第八代酷睿。而U系列处理器主要用于轻薄笔记本，可以在功耗和性能两者之间达到一个良好的平衡。在具体参数方面，Intel Core i5-8250U采用14nm的制作工艺，架构为Kaby Lake-Refresh，拥有4核心8线程，默认主频为1.6GHz，3级缓存为6MB，热设计功耗为15W。相较上一代i5-7200U，这一代的i5处理器增加了2个核心，3级缓存也有所提升，另外热设计功耗和更高一档次的i7-8550U一致。　　　　为了可以看出Intel Core i5-8250U的性能到底如何，我们用了Cinebench R15进行了跑分测试，它是很有说服力的一套CPU和显卡测试系统，最多能够支持256个逻辑核心，此外新版本还加强了着色器、抗锯齿、阴影、灯光以及反射模糊等的考察，对CPU性能的检测更加准确。　　从测试结果可知，Intel Core i5-8250U多核处理能力得分为526cb，而单核得分为137cb。而上一代的i5-7200U的测试结果为多核分数327cb，单核为90cb。单纯从数值上就可以看出，第八代酷睿处理器的性能已经有了很大的提升。　　　　显卡对电脑的性能也有着非常大的影响，为了保持轻便和节约能耗，这款Acer Swift 5搭载的是Intel UHD Graphics 620集成显卡。它是一个集成的图形单元，可被Kaby Lake中的各种超低电压处理器发现。它的时钟频率与HD Graphics 620类似，必须访问主存储器。由于这款显卡的功耗比较低，所以这台机器经过单次充电后，可连续播放10小时的本地4K UHD视频。　　　　那么Intel UHD Graphics 620到底能做些什么呢？选择了轻薄是否就意味着性能不足了呢？别急，我们运用了3D Mark这款软件进行了测试，这个是专业测试显卡性能的软件。我们选择的测试模式是Sky Diver，这个模式主要适用于游戏手提电脑和中端PC，可以测试主流显卡、移动设备GPU 或集成显卡的 DirectX 11 系统。Intel UHD Graphics 620在这个模式下的得分为3461，显卡分数为3278，物理分数为5524。总体来说，这样的分数可以应对日常工作。　　　　硬盘的读写速度也非常影响电脑的性能，如今SSD可谓是笔记本的标配了，既能保障读写速度，又能兼顾轻薄，所以Acer Swift 5配备了一块256GB的SSD。AS SSD Benchmark是一款SSD的专用测试软件，可以测试连续读写、4K对齐、4KB随机读写和响应时间的表现，并给出一个综合评分。从测试结果看出，Acer Swift 5的SSD综合得分为983，已经达到了现在固态硬盘的主流水平。另外还要说的是，这款Acer Swift 5配备了双固态硬盘位，最高可以拓展1TB的SSD，解决容量不够的问题。　　　　为了可以考量这款Acer Swift 5的综合性能，我们用PCMark 10做了专门的测试。PCMark 10可以衡量各种类型PC的综合性能，无论是在专业人士手中，还是属于普通用户，都可以通过这款软件更清晰地了解自己的电脑，从而让它发挥最大的性能。这款机器的整体得分为3174，其中常用基本功能得分为6984，生产力得分为5060，数位内容创作得分为2457。这样的得分达到了主流商务本的水平，用它处理日常工作是游刃有余的。　　　　续航能力也很重要，尤其是对于轻薄本来说，你往往要背着它出差旅行或者参加各种会议。Acer Swift 5官方给出的续航时间为8小时，AIDA64 Extreme检测显示它的电池容量为35951mWh。那么实际的表现又是如何呢？通过AIDA64的测试，我们用了PCMark 8对Acer Swift 5的续航能力做了专项测试，它的续航时间达到了2小时42分钟，整体表现比较出色，达到了轻薄本的主流水平。值得一提的是，在续航能力测试的过程中，笔记本一般都处于高负荷的工作状态，但是生活工作中这样的情况并不常见，如果仅仅用来常规办公，续航时间应该更接近于官方数据。　　　　　　文件压缩是大家经常用到的功能之一，虽然操作简单，但是却很考验电脑的性能，这次我们对一个大小为3.85G的文件夹进行了压缩，里边的文件比较多，所以对电脑也是一个考验。整个过程耗时大概为6分4秒，CPU的使用率在80%-90%徘徊。这样的表现说明Acer Swift 5可以轻松的完成压缩任务，没有什么压力。　　　　图像处理是我们工作中经常使用到的一项功能。这次我们用了Adobe Photoshop CC 2017这款家喻户晓的图像处理软件进行了测试，我们对一张单反照片加了几个滤镜特效，从AIDA64 Extreme的监测数据可以看出，在进行滤镜处理的过程中，CPU的使用率为40%左右。所以Acer Swift 5是可以胜任图像处理工作的。　　　　视频剪辑也是我们经常使用到的功能，虽然视频剪辑是一些生产力电脑才能胜任的工作，但是也依然不会妨碍到我们对电脑性能进行考察。我们用了Adobe Premiere Pro CC 2017这款知名的剪辑软件对Acer Swift 5进行了测试，我们对一段1080P的视频加了RGB曲线、均衡等简单的特效，渲染过程大概用了5分20秒左右，内存占用率为50%左右。所以Acer Swift 5进行简单的视频剪辑还是没有问题的。　　　　　　这款宏碁蜂鸟Acer Swift 5令人惊奇的地方还是很多的。它在轻薄方面下足了功夫，拥有970g的重量和14.9mm的厚度，这样身材在笔记本行列中可以说是非常身轻如燕了。另外，镁锂合金的设计材质为机身增加了陶瓷质感，让Acer Swift 5透露出来了轻奢气质。为轻薄独特设计的Intel Core i5-8250U处理器，让它在能耗和性能方面有了一个良好的平衡。长达8小时的续航时间，非常适合商旅人群。在日常应用方面，也可以满足办公所需。总体来说，Acer Swift 5是一款非常走心的轻薄本。
